  • Description: mSisAp is a mobile application that simplifying the process for machine operators and data processors to report the end results of cable production to the company. One of the valuable features we're incorporating is a barcode scanning function, which allows easy data entry stored in a secure database. This enhancement proves to be incredibly beneficial for companies.

Problem

Machine and data management efficiency has always been a long-standing challenge in the industrial and construction sectors, particularly in the cabling industry. The main constraint is the lack of real-time communication between operators, employees, and supervisors, resulting in delays in addressing machine issues, suboptimal equipment utilization, and ineffective data management.

Solution

To address these issues, the development of a mobile application has become one of the solutions to bridge the communication gap and enhance real-time machine and data management. This innovative approach involves using barcode scanning with a camera and integrating with a database API for monitoring and managing data in real-time. The goal of this mobile application is to assist operators in quickly reporting equipment conditions and activities, facilitate swift responses, and reduce potential problems.

Methodology

Implemented the Scrum methodology in the mobile application development process, encompassing team formation, appointment of a project master, determination of process timelines, selection of a product owner, creation of a product backlog, initiation of sprints, and sprint closure.

Architecture Code

The project adopts a modular code architecture within the Flutter Dart programming framework, featuring a well-organized folder structure. It includes dedicated folders such as Controllers, responsible for handling API calls and services; Views, housing UI components; Utils, serving as a centralized hub for managing global variables and session management.
