Description

This app provides a platform to facilitate discussions in farming forums, access agricultural articles, stay updated with the latest news, engage in farming consultations, and browse a list of agricultural extension workers. The application is available on both Android and iOS platforms, developed using Flutter and Dart with BLoC as its state management solution.

Methodology

The Agile methodology is applied in the development of the application to ensure iterative and collaborative processes, allowing for adaptive planning, continuous improvement, and flexibility in addressing user requirements. This approach ensures a smooth and efficient delivery of updates and features.

Architecture Code

The project adopts a modular code architecture within the Flutter Dart programming framework, featuring a well-organized folder structure. The state management is implemented using the BLoC (Business Logic Component) architecture, ensuring maintainable and scalable code.
