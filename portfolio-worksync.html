  • Description: WorkSync is a mobile application that revolutionizes organizational management by seamlessly integrating cutting-edge features such as face recognition and location tracking for attendance, a streamlined request management system for leave and overtime, enhanced user experience through multi-language support, and customizable themes, providing businesses with an all-encompassing solution to optimize efficiency and communication.

Problem

The longstanding challenge of efficiently managing employee attendance and activities in industries addressed through the use of physical facial recognition tools. However, this method becomes inefficient when a large number of employees need to use such devices, leading to queues, and delays. Secondly, the inefficiency in recording warehouse production results for delivery orders persists with manual recording methods leading to time-consuming double work and manual transfer to Excel.

Solution

Given these limitations, the goal of developing this mobile-based application is to create an alternative that facilitates access for all employees in recording attendance and warehouse production for delivery orders using their smartphones, leveraging the ubiquity of mobile devices, ensuring ease of use anywhere.

Methodology

The Waterfall methodology applied in the development of the application to ensure smooth analysis, design, implementation, testing, and distribution processes, resulting in an application that enables employees to easily attendance record, enhancing operational flexibility, and efficiency.

Architecture Code

The project adopts a modular code architecture within the Flutter Dart programming framework, featuring a well-organized folder structure. It includes dedicated folders such as Controllers, responsible for handling API calls and services; Pages, housing UI components; Modules, specifically designed for machine learning functionalities; and Utils, serving as a centralized hub for managing global variables, localizations, and session management.
