  • Description: DiskonKuyy is a web application providing a comprehensive list of discounts from various places, including restaurants and cafes.

Problem

Consumers often face challenges in efficiently searching for diverse discounts and comparing prices across various platforms, compounded by the inconvenience of navigating through different accounts and applications.

Solution

DiskonKuyy serves as a centralized platform, aggregating and verifying comprehensive discount information, empowering restaurant and cafe partners to input and manage their own promotions, ensuring accurate and up-to-date offers for users.

Architecture Code

The project adopts a clean code architecture within the Bootstrap (HTML, CSS, Javascript) programming, featuring a well-organized folder structure.
